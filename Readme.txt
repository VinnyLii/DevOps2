Time Management Application

This is a simple time management application created using Python. The application allows you to add courses and assignments, mark assignments as completed, and view your schedule.

Requirements

Python 3.6 or higher
setuptools
pytest

Installation

Clone or download the repository from Github.
Navigate to the project directory in your terminal or command prompt.
Run the following command to install the application:

python setup.py install

Usage

To run the application, navigate to the project directory in your terminal or command prompt and run the following command:

python TimeMgt.py

This will start the application and display a menu with options for adding courses and assignments, marking assignments as completed, and viewing your schedule.

Testing

To run the unit tests for the application, navigate to the project directory in your terminal or command prompt and run the following command:

python setup.py test

This will run the unit tests using pytest and display the results.

Packaging

To create a deployable package for the application, navigate to the project directory in your terminal or command prompt and run the following command:

python setup.py sdist

This will create a .tar.gz file in the dist directory that can be distributed or deployed to a server.
